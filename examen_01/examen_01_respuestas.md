# Examen 01 - Respuestas Teóricas

## 1. Define clase
Una clase es una plantilla o instruccion que define la forma o el comportamiento de un objeto.
- **Atributos:** Las características(variables).
- **Métodos:** Las acciones o comportamientos (funciones).

## 2. Define encapsulamiento
El encapsulamiento es el principio de ocultar los detalles internos de funcionamiento de un objeto y proteger sus datos. Se logra mediante modificadores de acceso:
- **Private:** Los datos solo son visibles dentro de la misma clase.
- **Public:** Se define una interfaz para que el exterior se muestre.

## 3. Define constructor
Un constructor es un método especial que:
- Tiene el mismo nombre que la clase.
- Se ejecuta automáticamente al crear un objeto.
- Su función principal es inicializar los atributos del objeto con valores válidos.
